# Sample Audio Files

Place your audio files in this directory.

## Supported Formats

- **MP3** (.mp3) - Most widely supported
- **WAV** (.wav) - High quality, uncompressed
- **OGG** (.ogg) - Open source format
- **AAC** (.aac) - Advanced Audio Coding
- **FLAC** (.flac) - Lossless compression

## Usage

1. Add your music files to this folder
2. Use the "Add Track" button in the application to upload them
3. Or reference them directly in the code

## Note

Large audio files are excluded from Git by default (see `.gitignore`).
This keeps the repository size manageable.

## Example Structure

```
audio/
├── pop/
│   ├── song1.mp3
│   └── song2.mp3
├── rock/
│   ├── track1.mp3
│   └── track2.mp3
└── sample.mp3
```
