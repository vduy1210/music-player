# Images Directory

Store all image assets here.

## Suggested Organization

```
images/
├── album-covers/      # Album art images
├── icons/             # UI icons
├── backgrounds/       # Background images
└── logos/             # Logos and branding
```

## Recommended Formats

- **PNG** - For images with transparency
- **JPG/JPEG** - For photographs and complex images
- **SVG** - For icons and scalable graphics
- **WebP** - Modern format with better compression

## Image Optimization

For best performance:
- Compress images before adding
- Use appropriate dimensions (avoid oversized images)
- Consider using WebP format for modern browsers
- Use lazy loading for multiple images

## Example Usage

```html
<!-- Album cover -->
<img src="assets/images/album-covers/cover1.jpg" alt="Album Cover">

<!-- Icon -->
<img src="assets/images/icons/play-icon.svg" alt="Play">
```
